## 每课产出规范

每课一个独立文件夹，包含以下文件：

```
第N课-课题名/
├── 讲义.md          # 讲课用讲义（对标《动态规划》笔记风格）
├── 模板代码.cpp      # 核心算法模板代码
└── 例题-*.cpp       # 经典例题代码（含注释）
```

## 技术规范

- **C++标准**：C++17，`using namespace std;`
- **代码风格**：`vector` 优先于 C 数组，`typedef long long ll;`
- **讲义格式**：Markdown，包含基础概念→例题分析→深度思考→代码实现→习题

## 备课流程

1. 读取此 INDEX.md 确定下一课
2. 对照《CSP算法篇课程体系框架》中对应专题的教学要点
3. 按产出规范生成全部文件
4. 更新本文件状态标记
